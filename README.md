# Home Care Harmony

Small e-commerce starter for Indian home-care products.

## How to run locally

Ensure you have Node.js (recommended LTS) and npm installed, then:

```bash
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Project info

- **Name**: Home Care Harmony
- **Description**: Sustainable and affordable home-care essentials tailored for Indian households.

## Technology stack

- Vite
- TypeScript
- React
- Tailwind CSS

## Deploy

Build with:

```bash
npm run build
```

Then serve the `dist` output using your preferred static host.

## Custom domain

Point your domain to your hosting provider following their docs. This repo does not include any provider-specific deployment automation.
